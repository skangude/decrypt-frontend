Decrypter Using Reference Text

This app allows one to upload a reference plain text file, which is used to decrypt an encrypted file using character and n-gram frequency analysis.

    The encrypted text should have the following characteristics:
  • The encryption should be a simple character mapping encryption called Simple Substitution Cipher.
  • The language characteristics of the reference plain text file should be the same that of the file that is encrypted. Similar characteristics implies similar character and n-gram frequencies at a high level.
